# Supporting Material for "A retrieval-augmented LLM framework for PRISMA-aligned systematic reviews: a case study on action quality assessment"

This repository contains supporting material for the manuscript currently under review.

## Data Availability

The file `DOI_list` provides the list of DOIs corresponding to the papers included in the analysis. <br>
For each entry, associated metadata (title, authors, year, data sources) are provided. <br>
Full-text papers are not redistributed in order to comply with copyright regulations.

## Code Availability

The code used during the current study is available from the corresponding author on reasonable request. <br>
The full codebase of the proposed framework, including the scripts used to generate the results and tables reported in the manuscript, will be made publicly available upon acceptance of the manuscript.
